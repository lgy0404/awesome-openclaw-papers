# Awesome OpenClaw Papers [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated collection of academic papers, benchmarks, and resources for the [OpenClaw](https://github.com/openclaw/openclaw) ecosystem — the open-source, self-hosted AI assistant platform.

## 🌐 [Browse the Full Collection →](https://lgy0404.github.io/awesome-openclaw-papers/)

The interactive website features search, category filtering, dark mode, and is always up-to-date.

## Add a Resource

We welcome community contributions! Two ways to add a paper or resource:

### Option A: Submit via Issue (easiest)

[Click here to submit](../../issues/new?template=add-paper.yml) — fill out the form and we'll take care of the rest.

### Option B: Edit & PR

1. Edit [`docs/_data/papers.yml`](docs/_data/papers.yml)
2. Add your entry following the format below
3. Submit a Pull Request — the website auto-rebuilds on merge

```yaml
- title: "Your Paper Title"
  description: >
    A brief summary of the paper's contribution.
  date: "2026-03"
  links:
    arxiv: https://arxiv.org/abs/XXXX.XXXXX
    code: https://github.com/...
  tags: [keyword1, keyword2]
```

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full guide.
